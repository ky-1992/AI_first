## 项目概述

**NeonatalDiary** 是一款面向新生儿父母的 Android 应用，用于记录宝宝每日的基本情况，包括：

- 喂养记录
- 情绪状态
- 体重监测
- 备注信息
- 图片/视频附件

### 技术栈

- **语言**: Kotlin 1.9.x
- **UI**: Jetpack Compose
- **架构**: MVVM + Clean Architecture
- **数据库**: Room
- **依赖注入**: Hilt
- **图片加载**: Coil
